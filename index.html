Exemplos de equações de recorrência

Torre de Hanói

T(n) = 2T(n-1) + 1

Representa o número mínimo de movimentos para resolver o problema da Torre de Hanói.

Progressão Aritmética

T(n) = T(n-1) + c

Cada termo é obtido somando uma constante ao termo anterior.

Progressão Geométrica

T(n) = aT(n-1)

Cada termo é obtido multiplicando o termo anterior por uma constante.

Substituição Linear

T(n) = aT(n-1) + b

Uma combinação linear do termo anterior multiplicado por uma constante e somando outra constante.

Expansão Binária

T(n) = T(n/2) + O(1)

Utilizada em algoritmos de divisão e conquista, como a pesquisa binária.
