Equações de Recorrência de Ordem 2

Fibonacci

T(n) = T(n-1) + T(n-2)

Cada termo é a soma dos dois anteriores, famosa sequência de Fibonacci.

Equação de Segunda Ordem Linear

T(n) = aT(n-1) + bT(n-2)

Representa uma combinação linear dos dois últimos termos da sequência.

Equação Homogênea

T(n) = aT(n-1) + bT(n-2)

Forma homogênea com coeficientes constantes.

Equação Não Homogênea

T(n) = aT(n-1) + bT(n-2) + f(n)

Equação de segunda ordem não homogênea, com termo dependente de n. Neste exemplo f(n) = n
